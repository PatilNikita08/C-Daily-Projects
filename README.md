# Day 1 – Eligibility Checker in C

## Description
This C program takes the user's name and age as input and checks
whether the user is eligible for driving a car and voting.

## Concepts Used
- Variables
- if-else condition
- scanf and printf
- String input

## Sample Output
Enter Your Name: Nikita  
Enter Your Age: 20  
Hello! Nikita  
You are eligible for driving a car and voting.
